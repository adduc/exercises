# Exercise: Building a simple GUI in Rust using FLTK

This exercise demonstrates how a small GUI application can be built in Rust
using FLTK. The application will display a simple window with a
button that, when clicked, updates a label with a message.

## Context

I have a Dell tablet with a limited amount of memory available, and am
interested in evaluating different solutions for building a lightweight GUI
application.

## Thoughts

FLTK used less than 20 MiB of memory (total RSS) without any tweaks out
of the box, which is great to see. It's default theming appears to be a bit dated.
